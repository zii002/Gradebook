The program builds a gradebook for the current semester, updates course grades, and combines it with the previous semester’s gradebook. It demonstrates how to work with lists, nested lists, indexing, and list methods in Python.

FEATURES
-Store subjects and grades in a nested list
-Add new courses to the gradebook
-Update grades
-Remove and replace values in lists
-Combine two gradebooks (previous semester + current semester)

CONCEPTS PRACTICED
-Python lists
-Nested lists
-Indexing
-List methods (append(), remove())
-List concatenation
